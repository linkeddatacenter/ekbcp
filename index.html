Welcome to EKBCP

EKBCP project provides an interactive control panel for LinkedData.Center APIs (eKB) and was initially developed by Turtle team during wikimedia 2016 in LinkedData.Center Campus in Esino Lario (Italy).

The project http://doc.yasgui.org/

EKBCP use YASGUI and 3Dsparql libraries integrated with the EKB APIs provided by LinkedData.Center

Have a look to our EKBCP service at http://cpanel.linkeddata.center/

Ekbcp is maintained by linkeddatacenter. This page was generated by GitHub Pages using the Cayman theme by Jason Long.
